# Background Music

Place `.mp3` or `.wav` background music files here.

The tool will randomly select one and auto-mix it at ~15% volume beneath the AI voiceover.

## Recommended Style
- Cinematic / suspense music
- No vocals (instrumental only)
- 60-120 seconds or longer (will be trimmed to match video)

## Free Sources
- https://freemusicarchive.org
- https://pixabay.com/music
- https://incompetech.com (Kevin MacLeod)
- YouTube Audio Library (royalty-free)
